# Claude Code 하네스 레이어 정리

| 레이어 | 위치 | 역할 | 언제 쓰나 |
|---|---|---|---|
| 메모리 | `CLAUDE.md` | 항상 로드되는 지속 지시문 | 프로젝트 규칙, 톤, 원칙 |
| 설정 | `.claude/settings.json` | 권한, 환경변수, 훅 등록 | 허용/차단 도구, 자동 실행 |
| 훅 | `.claude/hooks/*` + settings | 도구 호출 전/후 자동 실행되는 셸 | 포매터, 린터, 로깅 |
| 슬래시 커맨드 | `.claude/commands/*.md` | 사용자가 `/name` 으로 호출하는 프롬프트 | 반복 워크플로우 |
| 스킬 | `.claude/skills/<name>/SKILL.md` | 모델이 트리거어로 스스로 불러오는 절차 | 조건부 자동 워크플로우 |
| 서브에이전트 | `.claude/agents/*.md` | 격리된 컨텍스트에서 실행되는 전문가 | 탐색·리뷰 등 컨텍스트 무거운 일 |
| MCP | 외부 서버 | 추가 도구/데이터소스 | 외부 API 연동 |
